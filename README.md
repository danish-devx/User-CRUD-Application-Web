## 📌 About The Project

This is a **User CRUD Application Web** built using HTML, CSS, and JavaScript.  
It allows users to manage data easily with full **Create, Read, Update, Delete (CRUD)** functionality using a JSON server.

The project has a modern UI with responsive design and smooth user experience.

---

## ✨ Features

- ➕ Add New User
- 📄 View All Users
- ✏️ Edit User Information
- ❌ Delete User
- 📱 Fully Responsive Design
- 🎨 Modern UI with Animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- JSON Server (Backend API)

## 🚀 How It Works

1. Fetch user data from API
2. Display users in stylish cards
3. Add new user using form
4. Edit existing user data
5. Delete user instantly
6. UI updates dynamically without refresh
